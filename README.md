# -Customer-Shopping-Behavior-Analysis
🚀 End-to-End Data Analytics Project | Python | SQL | Power BI
📌 Project Overview

This project focuses on analyzing customer shopping behavior using a dataset of 3,900 transactions. The goal is to extract meaningful insights that can help businesses make data-driven decisions, improve customer engagement, and increase overall revenue.

The project follows a complete analytics lifecycle — from data preprocessing and exploration to visualization and business recommendations.

🎯 Business Objectives
Understand customer purchasing patterns
Identify high-value and loyal customers
Analyze the impact of discounts and subscriptions
Improve marketing and sales strategies
Enable data-driven business decisions
